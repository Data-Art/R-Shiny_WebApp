# R-Shiny WebApp
Visualize Web pages utilizing the shiny

Step 1 : Choose File(CSV or Text)

Step 2 : Choose Columns of Check Box

Step 3 : Select Chart and choose Chart type
* Pyramid Chart
* Bar Chart
* Line Chart
* Time Series Chart
